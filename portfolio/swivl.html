      Overview

      This application was designed to serve as the visual data analytics platform for closed environments. It was adaptable to a number of different services, providing an api to plug into.

      What I worked on

      My initial involvement with this project was a developer where I helped port the application from a KnockoutJS based frame to DurandalJS. Due to the need for more control over charts, we wrote our own charting library based on D3.js, moving from Highcharts.js. During this rewrite, I was involved with many of the discussions concerning the IA and interaction design of the project. I provided some visual design ideas as well.
